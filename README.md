# ⚡ Discipline_Engine_v3 | Mission Control

A high-performance, cyber-aesthetic habit tracking dashboard designed for precision discipline and academic productivity. Built with a "Mission Control" philosophy, this engine allows users to monitor their daily objectives through real-time telemetry and visual analytics.



## 🚀 Overview
The **Discipline_Engine_v3** is a frontend web application that transforms standard habit tracking into a mission-critical operation. It utilizes local browser storage to maintain data persistence without the need for a backend, making it fast, private, and secure.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism & Cyberpunk aesthetics)
* **Logic:** Vanilla JavaScript (ES6+)
* **Analytics:** [Chart.js](https://www.chartjs.org/) for real-time performance indexing.
* **Storage:** Browser `localStorage` & `sessionStorage`.
* **Fonts:** Google Fonts (Orbitron, Fira Code, Space Grotesk).

## 🛰️ Key Features
* **Unit Identification:** Custom callsign and profile picture upload for personalized mission deployment.
* **Mission Grid:** A 31-day interactive matrix to track objectives with binary (Pass/Fail) status.
* **Efficiency Telemetry:** Real-time circular progress tracking for daily completion rates.
* **Performance Index:** A dynamic line graph (Cubic Bezier) visualizing efficiency trends over the entire month.
* **Mobile Responsive:** Designed to be accessible across multiple terminal types (Phone, Tablet, Desktop).
